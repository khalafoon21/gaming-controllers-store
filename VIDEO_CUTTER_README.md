# 🎬 سكريبت تقطيع الفيديو

سكريبت بايثون متقدم لتقطيع الفيديو بالثواني المحددة مع إمكانية التحويل بين الوضع الرأسي والعامودي وتخطي حقوق الطبع والنشر.

## ✨ المميزات

- ✂️ **قص الفيديو بالثواني**: حدد وقت البداية والنهاية بدقة
- 🔄 **تحويل الاتجاه**: 
  - رأسي (أفقي - 16:9) للشاشات العريضة
  - عامودي (رأسي - 9:16) لمنصات التواصل الاجتماعي
- 🚫 **تخطي حقوق الطبع والنشر**: إزالة العلامات المائية تلقائياً
- 🎯 **وضعين للاستخدام**:
  - تفاعلي: سهل للمبتدئين
  - سطر الأوامر: سريع للمحترفين

## 📦 التثبيت

### 1. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

أو يدوياً:

```bash
pip install moviepy numpy imageio imageio-ffmpeg
```

### 2. تثبيت FFmpeg (إذا لم يكن مثبتاً)

**Linux:**
```bash
sudo apt-get install ffmpeg
```

**macOS:**
```bash
brew install ffmpeg
```

**Windows:**
قم بتحميل FFmpeg من [الموقع الرسمي](https://ffmpeg.org/download.html)

## 🚀 الاستخدام

### الوضع التفاعلي (سهل)

ببساطة قم بتشغيل السكريبت:

```bash
python video_cutter.py
```

سيطلب منك السكريبت:
1. مسار ملف الفيديو
2. وقت البداية والنهاية بالثواني
3. اختيار الاتجاه (رأسي/عامودي/بدون تغيير)
4. هل تريد إزالة حقوق الطبع والنشر
5. اسم الملف الناتج

### وضع سطر الأوامر (سريع)

#### مثال 1: قص بسيط
```bash
python video_cutter.py -i video.mp4 -s 10 -e 30 -o output.mp4
```
قص الفيديو من الثانية 10 إلى الثانية 30

#### مثال 2: قص وتحويل إلى عامودي
```bash
python video_cutter.py -i video.mp4 -s 5 -e 25 -o output.mp4 --vertical
```
قص الفيديو وتحويله إلى وضع عامودي (9:16) مناسب لـ TikTok, Instagram Reels

#### مثال 3: قص وتحويل إلى رأسي
```bash
python video_cutter.py -i video.mp4 -s 0 -e 60 -o output.mp4 --horizontal
```
قص الفيديو وتحويله إلى وضع رأسي (16:9) مناسب لـ YouTube

#### مثال 4: قص مع إزالة حقوق الطبع والنشر
```bash
python video_cutter.py -i video.mp4 -s 15 -e 45 -o output.mp4 --remove-copyright
```
قص الفيديو مع إزالة العلامات المائية

#### مثال 5: كل المميزات معاً
```bash
python video_cutter.py -i video.mp4 -s 10 -e 40 -o final.mp4 --vertical --remove-copyright
```
قص + تحويل عامودي + إزالة حقوق الطبع والنشر

## 📋 المعاملات

### المعاملات الأساسية:
- `-i, --input`: ملف الفيديو المدخل (مطلوب)
- `-s, --start`: وقت البداية بالثواني (مطلوب)
- `-e, --end`: وقت النهاية بالثواني (مطلوب)
- `-o, --output`: ملف الفيديو الناتج (مطلوب)

### المعاملات الاختيارية:
- `--vertical`: تحويل إلى وضع عامودي (9:16)
- `--horizontal`: تحويل إلى وضع رأسي (16:9)
- `--remove-copyright`: إزالة حقوق الطبع والنشر

## 💡 نصائح الاستخدام

### 1. معرفة مدة الفيديو
استخدم الوضع التفاعلي لعرض معلومات الفيديو أولاً

### 2. اختيار الاتجاه المناسب
- **عامودي (9:16)**: TikTok, Instagram Reels, YouTube Shorts
- **رأسي (16:9)**: YouTube, Facebook, Twitter
- **بدون تغيير**: للحفاظ على الأبعاد الأصلية

### 3. إزالة حقوق الطبع والنشر
السكريبت يقوم بقص 10% من أسفل الفيديو و 5% من الأعلى لإزالة العلامات المائية الشائعة

### 4. جودة الفيديو
الفيديو الناتج يحافظ على:
- نفس معدل الإطارات (FPS)
- جودة عالية (codec: libx264)
- الصوت الأصلي (codec: aac)

## 🎯 حالات الاستخدام

### للمحتوى على وسائل التواصل:
```bash
# فيديو TikTok (عامودي)
python video_cutter.py -i long_video.mp4 -s 30 -e 60 -o tiktok.mp4 --vertical --remove-copyright

# فيديو YouTube (رأسي)
python video_cutter.py -i long_video.mp4 -s 0 -e 120 -o youtube.mp4 --horizontal
```

### لاستخراج مقاطع محددة:
```bash
# استخراج أفضل لحظة
python video_cutter.py -i full_video.mp4 -s 125.5 -e 145.8 -o best_moment.mp4

# استخراج عدة مقاطع
python video_cutter.py -i video.mp4 -s 10 -e 20 -o clip1.mp4
python video_cutter.py -i video.mp4 -s 50 -e 70 -o clip2.mp4
python video_cutter.py -i video.mp4 -s 100 -e 130 -o clip3.mp4
```

## ⚠️ ملاحظات مهمة

1. **الأوقات بالثواني**: يمكنك استخدام أرقام عشرية (مثل: 10.5 ثانية)
2. **حجم الملف**: الفيديو الناتج قد يكون أصغر أو أكبر حسب الإعدادات
3. **الوقت المطلوب**: المعالجة تعتمد على طول الفيديو وقوة الجهاز
4. **الملفات المؤقتة**: يتم حذفها تلقائياً بعد الانتهاء

## 🐛 حل المشاكل

### خطأ: "FFmpeg not found"
قم بتثبيت FFmpeg كما هو موضح في قسم التثبيت

### خطأ: "No module named 'moviepy'"
قم بتثبيت المكتبات:
```bash
pip install -r requirements.txt
```

### الفيديو الناتج فارغ أو تالف
تأكد من أن أوقات القص صحيحة ولا تتجاوز مدة الفيديو الأصلي

### جودة الفيديو منخفضة
السكريبت يحافظ على الجودة الأصلية، لكن التحويل بين الاتجاهات قد يؤثر على الوضوح
